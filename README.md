# VLTS v46 - Astronomical Phenomena Educational System

## 综合课程中心 · Integrated Curriculum Center

这是一个天文主题的英语教育学习系统，集成了多种学习模块和趣味游戏。

### ✨ 主要功能

- **Spelling** 拼写训练 - 7种练习模式
- **Pronunciation** 语音训练 - 包含Phonics（262课）、IPA、Past Tense等
- **Test** 综合测试 - 8种考试模式
- **Funn** 趣味游戏中心 - 6个互动游戏
- **Reading Lab** 阅读实验室 - Fiction/Nonfiction资源库
- **Typing Dojo** 打字道场 - Story Mode + Music Mode
- **Word Bank** 词库管理 - Excel导入导出
- **Mistakes** 错题回顾
- **Reports** 学习报告

### 🎨 特色系统

- 🌌 太阳系动画 - 9大行星轨道运行，每个行星有独特音乐
- 🎵 BGM背景音乐 - 多主题切换，音乐淡入淡出
- 🕳️ 黑洞控制器 - 太极图腾控制器调节动画参数
- ✨ 粒子特效系统 - 流体粒子聚散动画
- 🍌 Nano Banana Pro - 趣味反馈系统
- 🌓 明暗主题切换 - 自适应配色系统
